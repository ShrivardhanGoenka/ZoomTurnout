How to set up your zoom account



1) Go to marketplace.zoom.us. Sign in with the account you will be using to host the meetings




2) After signing in, go to develop and then choose build app




3) Scroll Down a bit and choose the 'create' button in 'Webhook Only' section




4) Give your app any appropriate name('Attendance' is a good choice).




5) Fill in the required information and press 'continue'. This information is not relevant and you may put your own name in the 'company name' field.




6) This is the most important step. Please double check the following steps carefully

a) Switch on Event subscriptions by toggling the button

b) Select the 'Add new event subscription' button

c) Type 'Attendance' in the 'Subcription Name' and this exact url in the 'Event notification url' : http://zoomattendance.pythonanywhere.com/zoom/ . Please do not miss even the last forward slash

d) Select the 'Add events button' and add only the events depicted in the picture. Please select 'done' after adding the event types.

e) Select the 'Save button'

f) Copy the verification token which appears after pressing save. Please copy this into a temporary file as you will be needing it little later.

7) You should be getting the following page. If you do not please review all the steps. Please note that if you ever want to deactivate the app, i.e. stop ZoomAttendance from getting data for your zoom account you can do so by pressing the 'deactivate' button.

8) Go to the ZoomAttendance website and sign up(login if you have created your account). Click on Profile and press on the link to change your verification token. Paste your verification in the text field button and press submit.



Your account should now be ready. You can test it by starting a meeting and going to 'Current meeting' page. If the meeting details is displayed on the window then it confirms that it is working. If it does not work you should try all the steps below:

  • Check the url given by you in the 'features' page in the zoom account page.

  • Check that all the 4 events are checked in the events section.

  • Copy-paste the verification token once again. Please ensure that you didn't add an extra character(even not a space).

  • If it still does not work then please contact us.
